项目名称：社团管理系统（毕业设计项目）
创作时间：2022.11
1.此项目为本人的毕业设计项目，是基于前端技术（vue）和后端技术（springboot）及mysql开发的一个实现了前后端分离的后台管理系统，其中后台管理系统的前端代码利用了element-ui框架，
修改为适应社团管理的一个后台。
2.社团管理系统分为社团管理员，社团社长，社团成员，游客等四个主要角色，
对不同的角色绑定了不同功能，后台有如：社团类型管理，社团申请管理，社团成员管理，社团费用管理，在线申请及留言等功能，这个项目的设计是为了能更高效便捷的使用计算机系统来管理社团。
